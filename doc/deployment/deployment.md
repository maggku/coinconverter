## Deployed Version

   - **Deployment**:
   1. Log into GitHub
   2. From the list of repositories on the screen, select **coinconverter**.
   3. From the menu items near the top of the page, select **Settings**.
   4. Scroll down to the **GitHub Pages** section.
   5. Under **Source** click the drop-down menu labelled **None** and select **Master Branch**.
   6. On selecting Master Branch the page is automatically refreshed, the website is now deployed.
   7. Scroll bak to the **GitHub Pages** section to retrieve the link to the deployed website.

 
   - **Live Version**: You can view the live website at [GitHub Pages Link](https://maggku.github.io/coinconverter/).



### How to Run This Project Locally

To clone this project from Gitpod, you will need:

1. A **GitHub** account.
2. The **Chrome** browser.

Then follow these steps:

1. Install the **Gitpod Browser Extension** for Chrome.
2. After installation, restart the browser.
3. Log into Gitpod with your GitHub account.
4. Navigate to the [Project GitHub repository](https://maggku.github.io/coinconverter/).
5. Click the green **Gitpod** button in the top right corner of the repository.
6. This will trigger a new Gitpod workspace to be created from the code in GitHub, where you can work locally.

To work on the project code within a local IDE such as **VS Code**:

1. Open the [Project GitHub repository](https://maggku.github.io/coinconverter/).
2. Under the repository name, click **Code** > **Clone** > Copy the repository URL.
3. Open **Git Bash** in your local IDE.
4. Navigate to the directory where you want to clone the repository.
5. Run the following command:
   ```sh
   git clone <https://github.com/maggku/coinconverter.git>
